# Redirector

This project is a URL redirection and meta information server.

## Features
- URL redirection
- Meta tag generation for social sharing
- Easy deployment

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/news436/redirector.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the server:**
   ```bash
   npm start
   ```
